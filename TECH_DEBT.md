# Technical debt

- MVP выполняет кластеризацию точек MapLibre на клиенте; переход на серверные vector tiles предусмотрен контрактом `/map/clusters`.
- HEIC принимается только если установленный `sharp` умеет декодировать конкретный файл; клиенту возвращается понятная ошибка конвертации.
- Уведомления бота отправляются синхронно из API. Перед промышленной нагрузкой нужно подключить очередь (BullMQ) к уже выделенному notification service.
- Базовая админ-панель использует список Telegram ID из окружения; для команды модерации потребуется ролевая модель и журнал входов.
- Публичные OSM-тайлы подходят для разработки, но production должен использовать собственный или коммерческий tile endpoint с допустимой нагрузкой.
