# ADR 001: монорепозиторий и доверенная граница API

Статус: принято.

React, API и бот развиваются в npm workspaces. DTO и enum импортируются из `@pinory/shared`. Клиент никогда не определяет доступность PRIVATE/FOLLOWERS — SQL API фильтрует записи до сериализации. Это уменьшает рассинхронизацию контрактов и исключает утечку скрытых записей через клиентские запросы.
