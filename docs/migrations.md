# План миграций

1. `001_initial.sql`: PostGIS, UUID, пользователи, настройки, подписки, места, категории, иконки, отметки, медиа, подборки, комментарии, уведомления, жалобы, приглашения, уникальные просмотры, аналитика и idempotency; пространственные и поисковые индексы.
2. `002_location_v2_foundation.sql`: выключенная в продукте основа для добровольных location sessions/samples, visit events и счётчиков посещений.

Миграции неизменяемы после публикации. Каждая следующая миграция получает возрастающий номер и выполняется транзакционно командой `npm run db:migrate`.
